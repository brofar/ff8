# Button Byte Map
This is a representation of which button is pressed on a controller/keyboard. The keys are split into 2 bytes.

## Byte 1
* L3 +1
* R3 +2
* L1 +4
* R1 +8
* Circle +16
* Triangle +32
* X +64
* Square +128

## Byte 2
* Select +1
* Start +8
* Up +16
* Right +32
* Up Right +48
* Down +64
* Down Right +96
* Left +128
* Up Left +144
* Down Left +192
